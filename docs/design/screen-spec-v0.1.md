## 2. 정보 구조(IA)

```
[로그인/가입] → [/onboarding/pin] → [홈]
    ├── 받을 돈 / 갚을 돈 합계
    ├── 다가오는 예정일
    └── 최근 활동
[채무 목록] — 필터·검색
[채무 상세] — 타임라인 · 상환 · 조정 · 공유 · 합의종료 · 보관
[상대]
[설정] — 계정 · 알림 · 잠금 · 로그아웃 · 계정 삭제
[공유 /s/:token] — PIN 게이트(선택) → 읽기 전용 타임라인
```

## 3. 화면별 요구

### 3.1 홈 `/`

| 요소 | 내용 |
|------|------|
| 요약 카드 | 받을 총액·갚을 총액·진행 건수 (`archived` 제외, N4) |
| 리스트 | 최근 ledger 활동 또는 예정일 임박 채무 |
| 빈 | CTA 「첫 채무 등록」 |
| FAB/버튼 | 채무 등록 |

### 3.2 채무 목록 `/debts`

| 요소 | 내용 |
|------|------|
| 필터 | 전체 / 빌려줌 / 빌림 / 진행중(`active`) / 완료 / 연체(`overdue`, active만) |
| 검색 | 상대명·사유 키워드 |
| 카드 | 상대명, 방향 라벨, 잔액, 상태뱃지(`display_label`: 완료·합의종료만 **`completed`일 때**, 연체), `due_on` |
| 음수 잔액 | 「초과 상환 N원」표시(X18) |

### 3.3 채무 상세 `/debts/:id`

| 요소 | 내용 |
|------|------|
| 헤더 | 상대, 방향, **계산 잔액**, 상태 라벨(완료 / 합의 종료 / 연체) |
| 타임라인 | 개설(원금)·조정·상환, `occurred_on` 순(P9) |
| 액션 | 상환, 조정, 공유, 편집, **합의 종료**, 보관 |
| archived | 상단 띠 + 「보관 해제」— 편집·ledger 차단(X7) |
| ledger 삭제 | 확인 다이얼로그 |

**합의 종료:** 확인 모달 → `agreement_closed=true`, 라벨 「합의 종료」(P5②).

### 3.4 채무 등록·수정

| 필드 | 등록 | 수정 |
|------|------|------|
| 상대 | 선택·신규 | 변경 가능 |
| 방향 | lent / borrowed | **불가**(P12) |
| 원금 | 필수 | **불가**(P3) |
| 발생일 | 필수, ≤오늘 | 가능 |
| 사유 | 필수 1~500자 | 가능 |
| 예정일 | 선택, ≥발생일 | 가능 |

### 3.5 상환·조정

| | 상환 | 조정 |
|---|------|------|
| 금액 | 양수, >0 | 양수·음수, ≠0 |
| 일자 | ≤오늘, ≥채무 발생일 | 동일 |
| 메모 | 선택 0~500 | **필수** 1~500(X19) |
| 초과 상환 | P6 확인 모달 | X9 잔액<0 확인 |

### 3.6 공유 `/debts/:id/share`

- 만료: 30/90/180/무제한(기본 90, P11)
- 익명 토글(P14), PIN 4~6자리(선택)
- 활성 링크 1개 — 재생성 시 **교체 확인**
- 복사·회수

### 3.7 공유 보기 `/s/:token`

- 로그인·앱 PIN **불필요**
- PIN 설정 시 PIN 화면 선행
- 타임라인·잔액·방향·(익명 시 상대명 숨김)
- 푸터: payClear 출처, 법적 효력 없음
- 만료/회수: 안내 문구(`SHARE_INVALID`)

### 3.8 잠금 `/lock`

- PIN 또는 WebAuthn/생체
- 5회 오류 → 5분 잠금
- 공유 URL(`/s/*`)에서는 **표시 안 함**

### 3.10 상대 상세 `/contacts/:id` (S6)

| 요소 | 내용 |
|------|------|
| 헤더 | 상대명·메모 |
| 요약 | 이 상대에 대한 받을/갚을 소계(연결 채무 합) |
| 목록 | 해당 상대의 채무 카드 → 상세 이동 |

### 3.11 설정

| 섹션 | 내용 |
|------|------|
| 계정 | 이메일·인증 상태, 로그아웃 |
| 알림 | Push·이메일, D-1/당일 09:00 KST, iOS 미지원 안내 |
| 잠금 | PIN 변경, 백그라운드 1/5/15분 |
| 위험 | 계정 삭제(30일 유예, P13) |
