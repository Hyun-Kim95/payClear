# ADR-001: v0.1 플랫폼으로 웹+PWA 우선

| 항목 | 내용 |
|------|------|
| 상태 | 확정 |
| 날짜 | 2026-06-04 |
| PRD | U1, §1.4 |

## 맥락

- 개인 채무 기록 앱, 빠른 검증 필요
- 알림·잠금·공유 링크 필수
- 네이티브 앱은 개발·배포 비용 큼

## 결정

**v0.1은 반응형 웹 + PWA**(manifest, service worker, 홈 추가, Web Push).

- iOS PWA Push 한계 → **이메일 리마인더를 v0.1 필수 폴백**으로 포함
- v0.2+ 필요 시 Capacitor/스토어 앱 검토

## 대안

| 대안 | 기각 이유 |
|------|-----------|
| 웹만(PWA 없음) | 홈 추가·푸시 이점 포기 |
| 네이티브 우선 | MVP 기간·비용 |
| 로컬만 | U2 클라우드 요구와 불일치 |

## 결과

- 프론트: SPA + PWA 플러그인
- 알림: Push + 이메일 이중
- Gate 3에서 iOS 실기 알림 동작 별도 체크
