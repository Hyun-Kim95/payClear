---

## 1. 인증·온보딩

- [ ] 이메일 또는 소셜 로그인 후 홈 진입
- [ ] 소셜만 가입 시 이메일 등록·인증 플로우 동작
- [ ] 최초 진입 PIN 설정(건너뛰기 없음)
- [ ] 백그라운드 5분 후 잠금, PIN/생체 해제
- [ ] `/s/:token` 공유 URL은 앱 잠금 **미적용**

---

## 2. 채무·상대 (F1, F2)

- [ ] 상대 등록·동일 이름 중복 허용
- [ ] 채무 등록: 빌려줌/빌림, 원금, 발생일, **사유 필수**, 예정일 선택
- [ ] 원금 0원·미래 발생일·due<발생일 **거부**
- [ ] 상세 타임라인에 개설(원금) 표시, 시간순 정렬

---

## 3. 상환·조정·잔액 (F3, F4, F5)

- [ ] 상환 등록 후 잔액 자동 감소(S2)
- [ ] 조정(±) 후 잔액 반영, 조정 사유 필수
- [ ] 잔액 0 → 자동 완료·라벨 「완료」(E4)
- [ ] 합의 종료(수동) → 잔액>0 가능·라벨 「합의 종료」
- [ ] 합의 종료 후 잔액 0이어도 「완료」로 **바뀌지 않음**(X17)
- [ ] 합의 종료 후 상환으로 active 복귀 시 **합의 종료 뱃지 숨김**(P5b·E2)
- [ ] 잔액 다시 0 + `agreement_closed` → completed·「합의 종료」복귀(E2)
- [ ] 초과 상환 확인 후 저장, 잔액 음수 표시(E3)
- [ ] ledger 삭제 후 잔액·상태 재계산(E1)

---

## 4. 목록·요약 (F6, F7, F8)

- [ ] 필터: 진행중/완료/연체( active만 연체)
- [ ] 검색: 상대명·사유
- [ ] 홈: 받을/갚을 합계, archived 제외
- [ ] 합의 종료·잔액>0 건 요약에 포함(N4)
- [ ] 상대 상세에서 연결 채무·소계 확인(S6)
- [ ] (선택) 홈 「이번 달 예정」목록

## 5. 공유 (F16)

- [ ] 비로그인 조회: 타임라인·잔액(S7)
- [ ] 익명·PIN·만료(기본 90일) 동작
