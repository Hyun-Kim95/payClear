# payClear

개인용 **채무관리** 웹 앱(PWA). 메모장 대신 빌려줌·빌림·상환 이력을 기록하고, 잔액 근거를 한 화면에서 설명할 수 있게 합니다.

## v0.1 범위 (요약)

- 로그인 + 클라우드 저장
- 채무·상대·상환·조정·타임라인
- 공유 링크(읽기 전용)
- 상환 예정 알림(Web Push + 이메일)
- PIN/생체 잠금

**포함하지 않음:** 실제 송금, PG, 이자, 사업자·세무, 협업 편집
